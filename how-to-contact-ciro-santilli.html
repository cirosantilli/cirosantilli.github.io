🔗
If you are a privacy freak or are going to tell Ciro state secrets, here's Ciro's GNU Privacy Guard public key: pubkey.gpg. You can also use Ciro's ProtonMail which is of form: "Ciro's GitHub username + protonmail.com". Notifications that messages were received in the Protonmail mailbox are sent an unencrypted mailbox which Ciro views regularly, but the message content itself is not. Ciro aims to maintain the highest security standards feasible on that account, including Multi-factor authentication, a unique password not-reused anywhere else, immediate message deletion after reading, and only opening it from trusted devices.

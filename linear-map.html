Linear map (linear operator)

| 🗖 nosplit | ↑ parent "Matrix multiplication" | 123, 137, 2
🗖 nosplit | ⇓ toc | ↑ parent "Matrix multiplication" | Wikipedia | 123, 137, 2
🔗
A linear function from X×X→Y where X and Y are vector spaces.
🔗
Basically a generalization of both:
  • matrix multiplication, with X=Y=Rn
  • dot product, which is a key example of X​=Y
🔗
The key advatnage over those concepts however is that it makes more sense in infinite dimensional spaces like Hilbert spaces.
🔗
Calling such functions "matrices" in infinite dimensions is stretching it a bit, since we would need to specify infinitely many rows and columns.
🔗
For example, the left side of the time-independent Schrodinger equation is a linear map.

Photon polarization

| 🗖 nosplit | ↑ parent "Photonics" | 183, 221, 1
🗖 nosplit | ⇓ toc | ↑ parent "Photonics" | Wikipedia | 183, 221, 1
🔗
The polarization state and how it can be decomposed into different modes can be well visualized with the Poincare sphere.
🔗
One key idea about photon polarization is that it carries angular momentum. Therefore, when an electron changes orbitals in the Schrodinger equation solution for the hydrogen atom, the angular momentum (as well as energy) change is carried out by the polarization of the photon!

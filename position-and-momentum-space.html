One way to think is: what is the definition of space space? It is a way to write the wave function ψx​(x) such that:
  • the position operator is the multiplication by x
  • the momentum operator is the derivative by x
And the, what is the definition of momentum space? It is of course a way to write the wave function ψp​(p) such that:
  • the momentum operator is the multiplication by p

Demos at: web-cheat/webpack/
🔗
Usage: 
cd web-cheat/webpack
npm install
npm run build
🔗
This will produce, under dist/ the following minimized files:
  • dist/index.html: from web-cheat/webpack/src/index.html. You can open it to see: 
    Hello webpack
    show on the browser. This was added from JavaScript.
  • dist/main.js: from web-cheat/webpack/src/index.js and anything in its import tree, e.g.:
    • web-cheat/webpack/src/main.js sass source. It gets embedded the the JavaScript output as a string, and the JavaScript then applies it to the page, making the font blue
    • lodash third party library
🔗
You can also run this test with the development server on http://localhost:9000: 
npm start
which uses unminimized outptus, and automatically push reloads the page whenever you change any of the input files!

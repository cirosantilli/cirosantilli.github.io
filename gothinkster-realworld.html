Ahh, you can't have new ideas anymore!
🔗
Basically puts together every backend with Front-end web framework.
🔗
Got the first working local setup on Ubuntu 20.10, Vue.js, Node.js Express.js with:
  • https://github.com/cirosantilli/node-express-realworld-example-app/tree/mongo4
  • https://github.com/gothinkster/vue-realworld-example-app
🔗
Devs are a bit too focused on monetizing the project through their "how to use this project" premium tutorial, and documentation could be better: just getting the hello world of the most popular backend with the most popular frontend is not easy... come on.
🔗
Important missing things from the minimum base app:
  • email notifications however as tested on the live demo: https://demo.realworld.io/#/
  • server-side rendering:
    • https://github.com/gothinkster/realworld/issues/423
    • https://github.com/gothinkster/realworld/issues/266
🔗
By default, the backends hardcode the upstream public data API: https://conduit.productionready.io/api and you have to hack that to match the backend.
🔗

  • the React setup failed as shown at: https://github.com/gothinkster/react-redux-realworld-example-app/issues/187
  • the Nest.js failed on Ubuntu 20.10 as per https://github.com/lujakob/nestjs-realworld-example-app/issues/19
  • randyscotsmithey/feathers-realworld-example-app
